## Rest API

First fetch a quote for a solana-devnet to solana-devnet transaction using USDC. For more see [the API documentation](https://docs.range.org/reference/get_quote#/) for this endpoint.

```shell
export RANGE_API_KEY=YOUR_API_KEY

curl -G "https://api.faraday.range.org/v1/transactions/quote" \                                         
  -H "Authorization: Bearer $RANGE_API_KEY" \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  --data-urlencode "from_chain=solana-devnet" \
  --data-urlencode "to_chain=solana-devnet" \
  --data-urlencode "from_asset=4zMMC9srt5Ri5X14GAgXhaHii3GnPAEERYPJgZJDncDU" \
  --data-urlencode "to_asset=4zMMC9srt5Ri5X14GAgXhaHii3GnPAEERYPJgZJDncDU" \
  --data-urlencode "from_address=61YLomPfeDF7Urpi6hK43sNcs1dNSH23d7ofFomsBqzn" \
  --data-urlencode "to_address=7oFwhPphVtfivJrb3WFAhXmELRzx8F3i4iWwmCXotM3v" \
  --data-urlencode "amount=1000000" \
  --data-urlencode "slippage_bps=50" \
  --data-urlencode "travel_rule_compliant=false" | jq .
```

  -H "Content-Type: application/json" \
  -d '{
