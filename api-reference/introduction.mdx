- Generate an API key at https://app.range.org (APIs page). It takes ~30 seconds.
- Add the header: `X-API-KEY: <your-api-key>` to all requests.
- Use the [Quickstart](../reference/quickstart) for a minimal end-to-end example, then dive into each API's introduction page.
