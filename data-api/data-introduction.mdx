# Introduction

The Range Data API offers unified access to on-chain data and analytics across multiple blockchains and protocols. It is organized into several key sections:

## Address Information
Fetch detailed information and transaction history for blockchain addresses, including labels, counterparties, balances, and payment activity.

## Network Information
Access network-wide metrics such as transaction volumes, active accounts, whale movements, and cross-chain transfers. Useful for macro-level analytics and monitoring.

## Protocol Analytics
Analyze DeFi protocols with endpoints for protocol stats, transaction details, and protocol-specific activity.

## Stablecoin Analytics
**New!** The stablecoins section delivers deep analytics on stablecoin activity, including:

- **DeFi Protocol Utilization:** Track stablecoin deposits, borrows, utilization rates, and liquidity in lending markets and DEXs.
- **Trading & Dominance:** Monitor trading volumes, TVL, and stablecoin dominance in DeFi.
- **User & Wallet Analytics:** Analyze active wallets, cohort breakdowns, holder concentration, and wallet activity trends.
- **Network & Geographic Intelligence:** Understand exchange flows, regional adoption, and fiat on/off-ramp activity for stablecoins.
